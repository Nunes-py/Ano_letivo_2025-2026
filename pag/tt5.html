Introdução ao CSS

CSS significa Cascading Style Sheets.
É utilizado para estilizar páginas web criadas com HTML.

Com CSS podemos definir cores, fontes, espaçamentos, layouts e animações.

Ele separa o conteúdo da apresentação, tornando os sites mais organizados e profissionais.

Seletores e Sintaxe

Sintaxe básica:

seletor {
 propriedade: valor;
}

Principais seletores:

  • Seletor de elemento
  • Seletor de classe
  • Seletor de ID
  • Seletor universal

Formas de utilização:

  • CSS Inline
  • CSS Interno
  • CSS Externo
